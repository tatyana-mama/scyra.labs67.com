Compatible Peripherals

Cameras
  • + Any IP camera with RTSP stream (1080p recommended)
  • + USB webcams (for basic setups)
  • + CSI cameras (direct connection to Jetson)
  • Recommended: 2-3 cameras per clinic (reception, operatory, entrance)
Audio
  • + USB microphone for voice assistant
  • + Speaker/sound bar for AI voice output
  • Recommended: directional mic for operatory, omnidirectional for reception
Smart Home / Automation
  • + Aqara smart home hub (door sensors, presence detection)
  • + Smart lighting (Zigbee/WiFi)
  • + Display screen for patient-facing info

Clinic Checklist

What your clinic needs before installation:

  • 1. Stable internet connection (for initial setup and updates only)
  • 2. One power outlet near the reception or server area
  • 3. Ethernet cable run to camera locations (or WiFi cameras)
  • 4. 30 minutes for initial setup and calibration
  • 5. Staff briefing (15 min) — we provide training materials
